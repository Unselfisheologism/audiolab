# **App Name**: AudioViz Studio

## Core Features:

- File Upload: Upload audio files (mp3, wav, etc.) and corresponding transcript files (txt, srt).
- Synchronized Display: Display the audio waveform and transcript in sync, using a default timing for each line in the transcript.
- Transcript Editor: Allow users to edit and correct the transcript.
- Template Selection: Provide pre-built video templates (phone call, song, audiobook) with customizable image links for background visuals.
- Video Customization: Allow users to customize video settings, like resolution, aspect ratio, and output format.
- Video Export: Export the customized audio visualization as a video file.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to convey professionalism and stability.
- Background color: Light Gray (#F0F0F0) for a clean and modern aesthetic.
- Accent color: Vibrant Orange (#FF9800) to highlight important actions and interactive elements.
- Clean and modern fonts like Open Sans for both the UI and the transcript display.
- Use consistent and recognizable icons for common actions like upload, edit, play, and export.
- The layout should be divided into clear sections for file upload, transcript editing, template selection, and export settings.
- Subtle animations when syncing and previewing audio visualizations.
